# omegateam-02-inventory

ระบบจัดการสต็อกร้านเครื่องเขียนแบบ command-line เก็บข้อมูลด้วยไฟล์ JSON

## ฟีเจอร์
- ดูรายการสินค้าทั้งหมด
- เพิ่มสินค้าใหม่
- รับสินค้าเข้าสต็อก
- จ่ายสินค้าออกสต็อก
- แสดงข้อความเมื่อไม่มีสินค้าในระบบ

## วิธีใช้งาน
1. เปิดเทอร์มินัลในโฟลเดอร์โปรเจกต์
2. รันคำสั่ง:
   python main.py

## ไฟล์สำคัญ
- main.py : เมนูหลักของโปรแกรม
- inventory.py : ฟังก์ชันจัดการข้อมูลสินค้า
- items.json : ไฟล์เก็บข้อมูลสินค้า

## ทีม
- Product Owner: นางสาวนริสรา ไกยสินธุ์
- Scrum Master: นายชินชนกนันทร์ พรมศรี , นายพชรดนย์ มูลสาระ
- Developers: นายเตชสิทธิ์ แก้ววิเชียร , นายกิตติธร รูปสะอาด , นายพงศกร ศรีวิเศษ
